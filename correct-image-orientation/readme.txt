A simple, easy-to-use plugin that corrects the orientation of images uploaded from an iPhone based on their EXIF metadata.

== Description ==

With the Correct Image Orientation plugin, there's no need to worry about your images being displayed in the wrong orientation on your WordPress site. Specifically designed to handle images uploaded from an iPhone, this plugin reads the EXIF metadata of each image and automatically corrects its orientation.

The plugin is incredibly easy to use - just install it, and it's ready to go. No configuration or complicated setup required.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/correct-image-orientation` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

That's it! The plugin will now automatically correct the orientation of any images uploaded from an iPhone.

== Frequently Asked Questions ==

= Does this plugin work with images uploaded from devices other than an iPhone? =

This plugin is specifically designed to correct the orientation of images uploaded from an iPhone. It may not work as expected with images uploaded from other devices.

= Does this plugin work with image formats other than JPEG? =

This plugin works best with JPEG images, as they typically contain the EXIF metadata needed to correct the image orientation.
